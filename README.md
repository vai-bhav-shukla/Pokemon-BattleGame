# Pokemon-BattleGame

A simple Java console-based Pokémon battle game created to practice core Java concepts.

## Features
- Basic Pokémon battle mechanics
- Turn-based combat
- Console interface

## Future Plans
- Add multiple Pokémon
- Different attack types
- Better battle system
- Pokémon selection
- Save game progress

## Purpose
This project was created as part of my Java learning journey and will continue to improve as I learn more.
